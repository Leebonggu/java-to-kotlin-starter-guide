## Any

- 자바의 Object 역할(모든 객체의 최상위)
- 모든 Primitive Type의 최상위도 Any
- Any 자체는 Null을 포함하지 않음
  - Any? 를 사용해줘야함
- Any
  - equals
  - hasCode
  - toString

## Unit
- 코틀린 유닛은 실제 존재하는 타입이라는 것을 표현

## Nothing
- 함수가 정상적으로 끝나지 않았다는 사실을 표현
- 무조건 예외, 무한루프 함수
